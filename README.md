# phpspa-example

A simple example of a PHPSPA

## Prerequisites

- PHP 8.3
- [Composer](https://getcomposer.org/)

## Installation

```bash
composer install
```

## Running the Application

```bash
composer start
```

This will start the development server. By default, it runs at [http://localhost:8000](http://localhost:8000).
